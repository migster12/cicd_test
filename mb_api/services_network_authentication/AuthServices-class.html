AuthServices

AuthServices class Null safety

Service for account related requests.

Contains service for logging in, reset password. All credentials are all encrypted using argon with added inhouse encrypting process. EncryptionAlgorithm.generateLoginCredentials Please check BiometricsServices.bioLogin for biometrics sign in. On top of encryption, authentication are can only made on verified devices. Device authorization can be made via AuthServices.authorizeDevice.
