ScreenTimeout

ScreenTimeout class Null safety

Handles screen timeout functionality.

As part of app's security features, the app will timeout or logout automatically once reach max session time validity. App will also timeout if app will be idle for a certain time. Session validity and idle timeout are configurable on server. Please check ApplicationAssetsResponse.sessionTokenValidityMinutes and ApplicationAssetsResponse.idleTime
